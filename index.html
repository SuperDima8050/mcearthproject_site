🌍 Что такое Project Earth?

Мы создали сервер для Minecraft Earth на базе готового Vienna API и модифицируем клиент, чтобы позволить вам подключаться к нашим серверам. Это позволяет всем желающим продолжить играть в игру после закрытия официальных серверов. Начни свою новую игру в Minecraft Earth с нами!

⚙️ Как это работает?

1

Используем Vienna API

Мы запускаем сервер на базе готового Vienna API - открытого сервера, сделанного на Docker для восстановления Minecraft Earth

2

Патчим клиент

Модифицируем приложение Minecraft Earth, чтобы оно подключалось к нашему серверу вместо официального

3

Ожидай авторизацию

Подожди, пока мы авторизуем тебя в сети 🙂

4

Скачай APK

Скачай модифицированное приложение Minecraft Earth из нашего Discord сервера

5

Наслаждайся игрой!

Включи ZeroTier One и запусти игру. Добро пожаловать обратно в Minecraft Earth! 😊

❓ Часто задаваемые вопросы

Это бесплатно?
Да! Project Earth полностью бесплатен для всех игроков.
Что такое Project Genoa?
Project Genoa - это открытый проект для воссоздания серверной части Minecraft Earth. Мы используем их готовое API для работы нашего сервера.
Где скачать APK?
Модифицированный APK доступен на нашем Discord сервере. Присоединяйтесь и следуйте инструкциям!
Нужен ли мне ZeroTier?
Да, ZeroTier необходим для подключения к нашему серверу. Это создаёт безопасную виртуальную сеть для игры.
